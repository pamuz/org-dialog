* Introduction
A minor mode for Emacs org mode that adds two block types, PROMPT and
ASSISTANT. Write a PROMPT block, execute it, and its contents plus
nearby context are sent to an LLM. The reply is inserted directly
after as an ASSISTANT block. The goal is fast, low friction, precise
interaction with LLMs while editing org files. Inspired by SolveIt
(https://solve.it.com/).

* Quickstart
1. Install org-dialog.el on your load-path.
2. Require it or enable via package setup.
3. Add dialog headers to your org file:
   #+begin_example
   #+DIALOG_MODEL: gpt-5.2-chat
   #+DIALOG_ENDPOINT: https://your.endpoint/v1/chat/completions
   #+DIALOG_API_KEY: ENV_VAR_WITH_API_KEY
   #+DIALOG_SYSTEM: Brief system prompt
   #+end_example
4. Enable the mode:
   #+begin_example
   M-x org-dialog-mode
   #+end_example
5. Insert a prompt block:
   #+begin_example
   #+begin_prompt
   Your question here
   #+end_prompt
   #+end_example
6. Place point inside the block and press C-c C-c.
7. An ASSISTANT block is inserted or updated below.
